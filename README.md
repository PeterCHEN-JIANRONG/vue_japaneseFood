# vue_japanesefood

日本料理電商
使用Vue3、webpack、NPM、vue router、元件化，開發電商網站前、後台

# 頁面

- [前台頁](https://peterchen-jianrong.github.io/vue_japaneseFood/#/)
- [後台頁](https://peterchen-jianrong.github.io/vue_japaneseFood/#/admin)

### 說明

- 前台不需登入，後台需要登入。
- 請將.env 檔，VUE_APP_PATH 替換成自己申請的。

#### 目錄結構

##### assets

- stylesheets
  - components 元件的 scss
  - helpers 自訂變數 scss
  - all.scss 做 scss 總管
